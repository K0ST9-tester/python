# Python Projects

This repository contains a collection of my Python projects, focusing on simulations, algorithms, and experiments

Each project is organized in its own folder and includes its own description

**Game of Life**
An implementation of Conway's Game of Life - a cellular automator where a grid of cells evolves over time based on simple rules


**COVID Simulator**
A simulation of virus spread in a population, modeling how infections propagate over time 


## Requirements

Most projects use:
- Python 3.x
- common libraries such as `numpy` and `matplotlib`
